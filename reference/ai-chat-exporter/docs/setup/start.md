## extension 설치

* 크롬 브라우저 주소창에 `chrome://extensions/` 입력
* 우측 상단 `개발자 모드` 켜기
* 좌측 상단 `압축해제된 확장 프로그램을 로드합니다.` 클릭 후 작업 중인 폴더 선택
