# User Manual

This guide explains how to use the jnj-obsidian-clipper Chrome extension.

## Getting Started
- Install the extension and load it in Chrome via chrome://extensions/.
- Use Ctrl+Shift+O to open the popup for clipping web content to Obsidian.

## Clipping Content
- Open the popup on a webpage.
- Select a template if needed, then click to save as a Markdown note in your Obsidian vault.

For settings, visit the options page to manage vaults and templates.
